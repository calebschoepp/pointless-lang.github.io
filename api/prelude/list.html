zip(a, b) link 
From two lists, get a list of tuple pairs of elems from each list in order 

For lists [a0, a1, a2, ...], [b0, b1, b2, ...], return the list
[(a0, b0), (a1, b1), (a2, b2) ...], with length limited by the length
of the shorter input list
